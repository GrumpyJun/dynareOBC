This folder contains the estimation examples.

The data files can be regenerated by running BoundedProductivityDataGeneration and UnboundedProductivityDataGeneration.

To run the estimation(s), run:

dynareOBC UnboundedProductivityEstimation estimation [EstimationFixedParameters=beta] [EstimationFixedParameters=gamma] [EstimationFixedParameters=beta#gamma] [OrderOverride=1]
dynareOBC BoundedProductivityExactEstimation EstimationDataFile=BoundedProductivityEstimation.xlsx nocubature [EstimationFixedParameters=gamma]
dynareOBC BoundedProductivityEstimation estimation nocubature [OrderOverride=1]
dynareOBC BoundedProductivityExactEstimation EstimationDataFile=BoundedProductivityEstimation.xlsx nocubature
