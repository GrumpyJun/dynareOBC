This folder contains the estimation examples.

The data files can be regenerated by running BoundedProductivityDataGeneration and UnboundedProductivityDataGeneration.

To run the estimation(s), run:

dynareOBC UnboundedProductivityEstimation estimation [OrderOverride=1]
dynareOBC UnboundedProductivityExactEstimation EstimationDataFile=UnboundedProductivityEstimation.xlsx [EstimationUpdateSparseCubatureDegree=51]
dynareOBC BoundedProductivityEstimation estimation nocubature [OrderOverride=1]
dynareOBC BoundedProductivityExactEstimation EstimationDataFile=BoundedProductivityEstimation.xlsx nocubature [EstimationUpdateSparseCubatureDegree=51]
