This folder contains the estimation examples.

The data files can be regenerated by running BoundedProductivityDataGeneration and UnboundedProductivityDataGeneration.

To run the estimation(s), run:

dynareOBC UnboundedProductivityEstimation estimation [EstimationFixedParameters=[beta|gamma|beta#gamma  EstimationMinimisationFunctions=FMinConWrapper]] [OrderOverride=1]
dynareOBC UnboundedProductivityExactEstimation EstimationDataFile=UnboundedProductivityEstimation.xlsx [EstimationFixedParameters=gamma EstimationMinimisationFunctions=FMinConWrapper] [EstimationUpdateSparseCubatureDegree=51]
dynareOBC BoundedProductivityEstimation estimation nocubature [OrderOverride=1] EstimationMinimisationFunctions=FMinConWrapper
dynareOBC BoundedProductivityExactEstimation EstimationDataFile=BoundedProductivityEstimation.xlsx nocubature EstimationMinimisationFunctions=FMinConWrapper [EstimationUpdateSparseCubatureDegree=51]
